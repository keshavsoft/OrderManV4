# OrderManV4

## Features

- Helps you establish CRUD for OrderMan

## Usage

1. copy .env.local file info paste into .env file.
2. check Data folder present or not.
3. for frontEnd run main.js in ForFrontEnd -> V1-> Builder.
4. run the surver(`npm run start`).
